
## Installation Guide
1. Clone the repository:
   ```bash
   git clone https://github.com/Arifur-Rahaman/softwrd-task.git
2. Change to the project directory:
   ```bash
   cd softwrd-task
3. Install the required dependencies:
   ```bash
   npm install
4. Run project on local:
   ```bash
   npm run dev
